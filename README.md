Event Manager Web App

A full CRUD web application that allows users to create, edit, and delete personal events. Events are stored persistently using the browser's LocalStorage.

Features
- Add new events
- Edit existing events
- Delete events
- Persistent storage with LocalStorage
- Clean and responsive user interface

Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
